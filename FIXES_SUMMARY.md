## 💡 Бонусные улучшения (опционально)

В `yandex_mapkit_router_service.dart` есть метод для сравнения всех типов маршрутов:

```dart
final routes = await YandexMapKitRouterService.buildAllRoutes(
  from: userLocation,
  to: serviceLocation,
);

// routes содержит все 3 типа маршрутов с информацией
for (final route in routes) {
  print('${route.icon} ${route.type}: ${route.distanceText}, ${route.durationText}');
}
```

Можно показать пользователю все варианты и дать выбрать оптимальный!
