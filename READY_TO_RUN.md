## 💡 ПОЛЕЗНЫЕ СОВЕТЫ

### Горячие клавиши во время работы:
- `r` - Горячая перезагрузка (Hot Reload)
- `R` - Полная перезагрузка (Hot Restart)
- `q` - Выход из приложения

### Добавьте алиасы в ~/.zshrc:
```bash
# Auto Service App
alias ascd='cd /Users/marcus/Documents/auto_service_app'
alias asrun='cd /Users/marcus/Documents/auto_service_app && ./menu.sh'
alias asdoc='open /Users/marcus/Documents/auto_service_app/START_HERE.md'
```

После этого просто пишите `asrun` из любой папки!
