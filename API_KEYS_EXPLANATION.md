# 🔑 Объяснение API ключей Яндекс

## 📍 Два разных сервиса — два разных ключа!

### 1️⃣ MapKit API Key (для карт)
- **Текущий ключ**: `20a40c6c-d27c-46b6-b96a-b4b6a4cb47ba`
- **Используется для**:
  - Отображения карт Яндекс
  - Встроенного индикатора пользователя
  - Управления камерой карты
  - Маркеров и объектов на карте
  - Жестов и взаимодействия с картой
- **Где получить**: https://developer.tech.yandex.ru/ → "Картография и навигация" → "Мобильный SDK MapKit"

### 2️⃣ Router API Key (для маршрутов) ⚠️
- **Текущий ключ**: `20a40c6c-d27c-46b6-b96a-b4b6a4cb47ba` (из исходного проекта)
- **Используется для**:
  - Построения маршрутов между точками
  - Получения детальной геометрии дорог
  - Расчета времени в пути
  - Расчета расстояния
  - Информации о маневрах (повороты, перестроения)
- **Где получить**: https://developer.tech.yandex.ru/ → "Маршрутизация и расчёт расстояний"

## 🚨 Проблема, которая была

```
I/flutter (12283): ⚠️ Routing API ошибка 401
I/flutter (12283): ⚠️ Яндекс недоступен, используем OSRM fallback...
```

**Причина**: Использовался ключ MapKit (`20a40c6c-...`) для Router API — это неправильно!

**Решение**: Использовать отдельный ключ Router API (`cb3b7ff7-...`)

## 📝 Как это работает

```dart
// lib/core/config/yandex_config.dart

class YandexConfig {
  // ✅ Для ОТОБРАЖЕНИЯ карт
  static const String mapKitApiKey = '20a40c6c-d27c-46b6-b96a-b4b6a4cb47ba';
  
  // ✅ Для ПОСТРОЕНИЯ маршрутов (ДРУГОЙ ключ!)
  static const String routingApiKey = '20a40c6c-d27c-46b6-b96a-b4b6a4cb47ba';
}
```

## 🔧 Если нужен свой ключ Router API

1. Зайдите на https://developer.tech.yandex.ru/
2. Войдите в аккаунт
3. Перейдите в раздел "Сервисы" → "Маршрутизация и расчёт расстояний"
4. Нажмите "Подключить"
5. Создайте новый ключ
6. Замените `cb3b7ff7-...` на свой ключ в `yandex_config.dart`

## 💡 Важные моменты

- ❌ **Нельзя** использовать один ключ для обоих сервисов
- ❌ **Нельзя** использовать MapKit ключ для Router API
- ✅ **Можно** использовать Router API ключ из исходного проекта (как сейчас)
- ✅ **Лучше** получить свой собственный ключ Router API

## 📊 Лимиты

### MapKit API
- Бесплатно: до 25,000 запросов в день
- Платно: от $10 за дополнительные запросы

### Router API
- Бесплатно: до 1,000 запросов в день
- Платно: от $0.005 за запрос
