## Функциональность

### Автоматическая загрузка демо данных

Провайдер теперь автоматически использует демо товары в следующих случаях:
- Когда API недоступен
- Когда API возвращает пустой список
- Когда возникает ошибка при загрузке с API

### Ручная загрузка демо данных

Также добавлен метод для ручной загрузки демо товаров:

```dart
await Provider.of<ProductsProvider>(context, listen: false).loadDemoProducts();
```

## Преимущества

1. **Тестирование без backend**: Можно тестировать функционал магазина без подключения к API
2. **Презентация**: Приложение всегда показывает товары, даже когда backend недоступен
3. **Разработка**: Разработчики могут работать с интерфейсом без зависимости от backend

## Технические детали

- Демо данные хранятся в статическом списке `_demoProducts`
- Все демо товары имеют корректные изображения с Unsplash
- Товары включают все необходимые поля: цены, рейтинг, количество и т.д.
- Данные сохраняются в локальное хранилище через SharedPreferences

## Использование

При инициализации приложения провайдер автоматически пытается загрузить данные с API, 
и в случае неудачи использует демо данные. Это обеспечивает бесперебойную работу 
приложения в любых условиях.
