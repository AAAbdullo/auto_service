## 🎬 Пошаговая анимация запуска

### 1️⃣ Откройте Terminal
```
Command + Space → "Terminal" → Enter
```

### 2️⃣ Перейдите в проект
```bash
cd /Users/marcus/Documents/auto_service_app
```

### 3️⃣ Запустите меню
```bash
chmod +x menu.sh
./menu.sh
```

### 4️⃣ Выберите действие
```
Появится меню:
╔════════════════════════════════════════╗
║   Auto Service App - Быстрые команды   ║
╚════════════════════════════════════════╝

1) 🚀 Запустить приложение
2) 🔧 Настроить проект с нуля
3) 🧹 Очистить и пересобрать
...

Введите номер: 1
```

### 5️⃣ Ждите запуска
```
🚀 Запускаем приложение...
✅ Симулятор уже запущен
🔍 Доступные устройства:
iPhone 14 Pro (simulator)
▶️  Запускаем приложение...
```

### 6️⃣ Готово!
```
🎉 Приложение запущено!
Используйте:
  r - для горячей перезагрузки
  R - для полной перезагрузки
  q - для выхода
```
